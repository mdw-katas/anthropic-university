# MATH 240 — Linear Algebra

**Prerequisites:** MATH 110

## Course Description

Vectors, matrices, and linear transformations — with a computational,
geometric emphasis rather than a proof-heavy one. Linear algebra is the
mathematics underneath computer graphics, machine learning, network
analysis, and much of scientific computing; CS 430 and 440 lean on this
course directly.

## Learning Objectives

- Compute with vectors and matrices: addition, scalar multiplication, dot
  products, matrix multiplication — and know what each *means* geometrically.
- Solve systems of linear equations with Gaussian elimination and interpret
  the solution set.
- Explain linear transformations and read a matrix as a function.
- Understand span, linear independence, basis, and dimension.
- Compute and interpret eigenvalues and eigenvectors.
- Connect the concepts to computing: least squares, PCA (preview), and why
  GPUs are matrix machines.

## Topic Outline

1. Vectors: geometry, arithmetic, dot product, norms, cosine similarity.
2. Systems of linear equations; Gaussian elimination; row echelon form.
3. Matrices as transformations; matrix multiplication as composition.
4. Matrix inverses; when and why a matrix is singular.
5. Vector spaces and subspaces; span, linear independence, basis, dimension,
   rank.
6. Orthogonality: projections, Gram-Schmidt, orthonormal bases.
7. Least squares: fitting a line/model to data (the gateway to ML).
8. Determinants: meaning as volume scaling.
9. Eigenvalues and eigenvectors; diagonalization; a look at PageRank.
10. Singular value decomposition (survey) and dimensionality reduction.
